8.1 Prismatic

  • Prismatic lets us play around with palettes by shifting the entire set at once. For example, starting with terrain..
terrain <- prismatic::colour(terrain.colors(10))
plot(terrain)

  • Turn the pallete grey
terrain_gray <- prismatic::clr_grayscale(terrain)
plot(terrain_gray)

  • Darken the pallete
terrain_darken <- prismatic::clr_darken(terrain)
plot(terrain_darken)

  • Make the pallete negative
terrain_negate <- prismatic::clr_negate(terrain)
plot(terrain_negate)
